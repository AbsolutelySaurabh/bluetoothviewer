essential for first release
---------------------------
- full code review
- background image
- script for release build, signed properly


release procedure
-----------------
set android:versionCode=1 (increasing integer)
set android:versionName=1.0.0
create release-1.0 branch on launchpad
switch to release branch
remove Log calls
remove android:debuggable
release build, signed properly
test well
upload to market
